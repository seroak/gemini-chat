## 지표 계산 방법

### 트리거율

```
트리거율 = (Rule이 트리거된 세션 수 / 총 세션 수) × 100
```

- alwaysApply Rule: 항상 100%
- globs Rule: 해당 파일을 다룬 세션 비율
- manual Rule: 사용자가 명시 요청한 세션 비율

### 유용률

```
유용률 = (✅ 유용 판정 수 / 트리거된 세션 수) × 100
```

- ⬜(무관)과 ❌(방해)는 유용하지 않은 것으로 계산

### 효율 점수

```
효율 점수 = 유용률(%) / Rule 줄 수
```

- 줄 수가 적으면서 유용률이 높을수록 효율적

### ROI 등급

```
A (필수):      유용률 80%+ & 트리거율 50%+
B (양호):      유용률 60%+ & 트리거율 30%+
C (검토 필요): 유용률 40%+ 또는 트리거율 낮지만 전문 영역
D (최적화):    유용률 40% 미만 또는 트리거율 10% 미만
F (삭제 후보): 3개월 연속 트리거 0% 또는 유용률 20% 미만
```

**참고**: alwaysApply Rule은 트리거율이 항상 100%이므로 유용률만으로 등급 결정.
